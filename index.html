基本概念

常用指令

  • git config --global user.name xxx
  • git config --global user.email xxx
  • git config --global --list 查看全局配置
  • git config -h 查看config指令的相关说明
  • git init 将一个文件夹初始化为git仓库
  • git status 查看仓库文件状态
  • git 文件名/. 将文件添加到暂存区（若之前文件没有被跟踪，此时及开始跟踪）
  • git commit -m '提交文本库的名'
  • git reset HEAD 取消暂存区的文件
  • git checkout --文件名
  • git checkout -b
  • git merge 分支名
